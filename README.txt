 GENERAL USAGE NOTES
---------------------
 - Requires a browser and an internet connection

 - Can be used and run under Windows or Linux

 
 Running under Windows
----------------------
In order to run the application under Windows you need to set up a 
localhost server for Windows(Apache, Python etc.), or connect to 
another user who is running the app locally and can share you his local IP

 Running under Linux
----------------------
Just double click on start.sh file
